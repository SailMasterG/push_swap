# push_swap

## Description

`push_swap` is a sorting algorithm project from the 42 curriculum. The goal is to sort a stack of integers using two stacks (A and B) and a restricted set of 11 operations, producing the shortest possible sequence of moves.

The program receives a list of integers as command-line arguments and prints to stdout the operations needed to sort stack A in ascending order.

**Sorting strategy:**
- 1 number: nothing to do
- 2 numbers: swap if out of order
- 3 numbers: dedicated sort (at most 2 operations)
- 4+ numbers: Radix sort — each number is assigned a binary index, then sorted bit by bit using `pb` (bit = 0) and `ra` (bit = 1), restoring all elements to A after each pass

**Available operations:**

| Operation | Description |
|-----------|-------------|
| `sa` | Swap the first two elements of stack A |
| `sb` | Swap the first two elements of stack B |
| `ss` | `sa` and `sb` simultaneously |
| `pa` | Push the top of stack B onto stack A |
| `pb` | Push the top of stack A onto stack B |
| `ra` | Rotate stack A upward (first element goes to bottom) |
| `rb` | Rotate stack B upward |
| `rr` | `ra` and `rb` simultaneously |
| `rra` | Reverse rotate stack A (last element goes to top) |
| `rrb` | Reverse rotate stack B |
| `rrr` | `rra` and `rrb` simultaneously |

**Performance:**

| Input size | Operations | Score |
|------------|------------|-------|
| 100 numbers | ~1300 | 2/5 |
| 500 numbers | ~7800 | 3/5 |

## Instructions

### Requirements

- GCC compiler
- GNU Make
- The `libft` library is included under `lib/libft/`

### Compilation

```bash
make
```

This produces the `push_swap` executable in the project root.

```bash
make clean   # remove object files
make fclean  # remove object files and executable
make re      # fclean + make
```

### Execution

```bash
./push_swap 5 3 1 4 2
```

Pass integers as separate arguments or as a quoted string:

```bash
./push_swap "5 3 1 4 2"
```

Generate a random sequence for testing:

```bash
./push_swap $(seq 1 500 | shuf | tr '\n' ' ')
```

### Checking correctness

Using the provided checker binary:

```bash
./push_swap 5 3 1 4 2 | ./checker_linux 5 3 1 4 2
```

Outputs `OK` if the sequence sorts the stack, `KO` otherwise.

### Error handling

The program prints `Error` to stderr and exits when given:
- Non-integer input (`"abc"`, `"12a3"`)
- Integer overflow (outside `INT_MIN` / `INT_MAX`)
- Duplicate values
- Empty or invalid tokens
